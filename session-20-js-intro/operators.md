# Operators

1. Assignment operator

- a=10 means 10 value stored in a
- a=b means b value stored inside a
- always right side value is stored inside left

2. Sorthand operators

- x = x+5 can we written as x+=5
- x = x-5 can we written as x-=5
- x = x*5 can we written as x*=5
- x = x/5 can we written as x/=5

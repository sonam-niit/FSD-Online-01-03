# Javascript Intro

- its a language used to add interactivity to HTML page
- we can use JS for Servcer Side as well as client side
- if its running from browser its client side
- If we are using Node JS to run JavaScript its called Server Side

## How to Setup Node JS

[Link to Download Node](https://nodejs.org/en/download)

- Download LTS version msi file for Windows
- double click on downloaded file and install by just clicking next next and finish
- once installation done successfully 
- one command prompt and enter below commands
- node -v (check Node Version)
- npm -v (check NPM version which install automatically when node installed)

## Create DataTypes

- for declaring variables in JS we are using let keyword
- variable name can start with letter and continue with numbers or _ or $
- you can not use special characters like . , % etc while giving name of variable
- naming convention: second word first letter capital
    - isLoggedIn, firstName, lastName
